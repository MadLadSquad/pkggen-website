A drop-in replacement for your entire desktop packaging and deployment pipeline

Launch fast

Plugins for popular platforms like GitHub are already included

Flexible

Whether you're generating 1 or 1000 packages, pkggen is designed to be as flexible as possible

Extensible

Our powerful plugin system allows you to easily customize pkggen to fit your needs

Simplistic interface

Go through the entire generation, testing and deployment pipeline in only 3 CLI commands

pkggen:
  generator: github
  packages:
	- myproject:
	  github:
		username: example
		repo: myproject
		query: releases
		tarballs: [ "{{ raw {{ pkgname }}}}.tar.gz" ]
